섹션 어휘 시각 사전

structure.md 가 페이지를 조립할 때 쓰는 고정된 섹션 어휘를 눈으로 익히는 사전입니다. 각 박스가 어떤 섹션인지 화살표로 표시했습니다.

!
이건 "모든 프로젝트가 이렇게 생겼다"가 아닙니다. 배치(순서·유무)는 프로젝트마다 다릅니다. 공통인 것은 "섹션의 종류(어휘)"뿐이고, 이 사전은 그 어휘를 시각적으로 익히기 위한 대표 예시 2개입니다. 실제 페이지는 structure.md 에서 이 어휘로 매번 새로 조립합니다.
A · 마케팅 페이지 예시

랜딩 페이지 = 마케팅 섹션 스택

상단 -> 하단으로 쌓인 각 박스가 하나의 섹션입니다.

->

->

->

"후기 인용"
->
Testimonialquotes[]{quote, author} · 신뢰
[ 지금 예약하기 ]
->
CTAheading · button · 행동 유도
(c) legal
->
Footercolumns[] · legal · 최하단
